# 🌳 AVL Tree Visualizer (Advanced UI)

An interactive and visually rich **AVL Tree Visualizer** built using **HTML, Tailwind CSS, and JavaScript**.  
This project demonstrates the working of a **self-balancing Binary Search Tree (AVL Tree)** with an advanced, modern user interface.

---

## ✨ Features

- ✅ AVL Tree insertion with automatic balancing
- 🔁 Supports all rotations: **LL, RR, LR, RL**
- 🎨 Advanced **glassmorphism UI**
- 📊 Dynamic SVG-based tree visualization
- 🧠 User-driven input (size + values)
- ⚡ Fast, client-side rendering (no backend)
- 🌐 Deployable using **GitHub Pages**

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **Tailwind CSS** – Styling & layout
- **JavaScript (ES6)** – AVL logic & rendering
- **SVG** – Tree visualization
